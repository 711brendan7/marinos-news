# iOS ショートカット セットアップ手順

## 全体の流れ

```
ホーム画面のショートカットをタップ
  → テキスト入力ダイアログ
  → Google Apps Script に送信
  → スプレッドシートに保存
```

---

## ステップ1：Google Apps Script のデプロイ

1. [script.google.com](https://script.google.com) を開き、**Google スプレッドシートに紐づく**新規プロジェクトを作成
   - スプレッドシートを新規作成 → 「拡張機能」→「Apps Script」
2. `todo/gas/Code.gs` の内容を貼り付ける
3. **`SECRET_TOKEN` を変更する：**
   - 例: `myTodo2024abc`（任意のランダム文字列）
4. 「デプロイ」→「新しいデプロイ」→ 種類: **ウェブアプリ**
   - 実行ユーザー: **自分**
   - アクセスできるユーザー: **全員**（匿名を許可）
5. 「デプロイ」ボタン → **ウェブアプリの URL** をコピーして保存

---

## ステップ2：Web アプリの初期設定（スマホ）

1. GitHub Pages の URL をスマホブラウザで開く
2. 初回起動時に「セットアップ」画面が表示される
3. **GAS デプロイ URL** と **SECRET_TOKEN** を入力して「保存して開始」
4. ホーム画面に追加（Safari → 共有 → ホーム画面に追加）

---

## ステップ3：iOS ショートカットの作成（素早い入力用）

「ショートカット」アプリを開き、「+」で新規作成。

### アクション一覧

**① テキストを入力**
- アクション名: `テキストを入力`
- プロンプト: `TODO を追加`
- 結果を変数 `TODO本文` に保存

**② JSON を組み立てる**
- アクション名: `テキスト`
- 内容（そのままコピー）:
  ```
  {"action":"add","text":"[TODO本文]","token":"SECRET_TOKEN_の値"}
  ```
- 結果を変数 `JSON本文` に保存

**③ GAS に送信**
- アクション名: `URLの内容を取得`
- URL: `（ステップ1でコピーしたウェブアプリ URL）`
- メソッド: `POST`
- リクエスト本文: `JSON本文`
- ヘッダー: `Content-Type: application/json`

**④ 結果を通知**
- アクション名: `通知を表示`
- 内容: `「[TODO本文]」を追加しました`

### ホーム画面への追加

1. ショートカットの「…」→「ホーム画面に追加」
2. アイコンと名前（例: `TODO追加`）を設定

---

## 使い方まとめ

| 操作 | 方法 |
|------|------|
| TODO を確認・管理 | Web アプリを開く（ホーム画面のアイコン） |
| TODO を素早く追加 | iOS ショートカットをタップ → テキスト入力 |
| 完了にする | Web アプリでチェックマークをタップ |
| 削除する | Web アプリで「×」をタップ |

---

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| "Unauthorized" エラー | トークンが `Code.gs` の `SECRET_TOKEN` と一致しているか確認 |
| 読み込みに失敗 | GAS URL が正しいか、デプロイが「全員」アクセスになっているか確認 |
| 追加したのに表示されない | ページを再読み込み（プルダウン） |
